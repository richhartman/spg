Avenging Kenny's Death
Korobrown Software

An often name-checked game as one of the first South Park games to be ever released online (the other game referred to as the first game ever made would be Nomad's South Park), this is a quite interesting early creation that manages to mesh several game genres together, ranging from platform-like action to an arcadey whack-a-Cartman game (see screenshot). In the game, Cartman has killed Kenny and it's Stan's job to make sure that Kenny is avenged by punishing his fat murderer. It features some really weird graphics at parts as some of the boys' sprites have totally wrong colors, but the rest of the graphics shine for such an early game. It also features well-done programming that pushes Klik and Play's limits. It might come of as a basic game today, but this remains one of the better games of this era, which cherishely marks South Park Gaming's modest debuts.
